# The Quiet Between

The Quiet Between is a custom Minecraft mod developed in Java using MCreator.
The project focuses on event-driven game logic, custom entities, and player interaction
within an existing game engine API.

## Project Goals
- Explore mod development using Java and Gradle
- Work with an event-based system inside a large codebase
- Apply object-oriented design in a game environment

## Key Features
- Custom entities and behaviors
- Player-triggered events and conditions
- Modular logic implemented through Java classes and procedures

## Technical Details
- Language: Java
- Build System: Gradle
- Modding Framework: MCreator (used as a development framework)
- Target Platform: Minecraft Java Edition

## Project Status
This project is currently under active development and is considered an early-stage prototype.

## Repository Structure
- `src/` – Java source code
- `elements/` – MCreator element definitions
- Gradle files – Build and dependency management

## Build / Run
The project can be built using Gradle after opening it in MCreator or a compatible Java environment.
